True or False: Using Linux is sufficient to ensure that the Open Source requirement of Open Science projects is met.
True
False
False! Performing analysis from a Linux can contribute to the Open Source requirement, but it alone does not suffice. Every software artifact used within an Open Science project must be Open Source and made available to researchers in order for an Open Science project to be considered Open Source.
